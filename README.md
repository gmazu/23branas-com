# 23branas-com

Sitio web de entrada para `23branas.com`.

Este repositorio contiene una version independiente de la landing principal, separada de `3Branas-com` para mantener historial, dominio y publicacion propios.

## Que contiene hoy

- `index.html`: entrada principal
- `en/index.html`: version inglesa
- `es/index.html`: version en español
- `assets/img/`: imagenes optimizadas para el sitio
- `assets/icons/`: favicon e iconos

## Direccion del sitio

La web funciona como una entrada visual y conceptual al proyecto:

- figura central de colision
- lectura tipo paper
- panel visual principal como parte del contenido, no como decoracion
- estructura bilingue

## Notas de trabajo

- El panel central de colision en `en/index.html` y `es/index.html` forma parte de la lectura visual principal del sitio.
- Las imagenes externas en `/home/gmazuel/www/img` deben revisarse manualmente antes de integrarse a la web.
- No forzar nombres fisicos o cosmologicos en nuevas imagenes hasta confirmar su lectura.
